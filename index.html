Publications

Efficient processing of large-scale sparse matrix-matrix multiplications on a single machine

Oct. 2017, IEEE SMC

Yong-Yeon Jo, Kyuhwan Lee, Myung-Hwan Jang, Sang-Wook Kim, Eunjee Song

This paper addresses how to perform two sparse matrices multiplication (SpGEMM) efficiently on a single-machine-based graph engine. We formulate the cost models for the three matrix product types (i.e., inner products, outer products, and row-row products) by reflecting their characteristics and propose memory allocation schemes.

Analysis on Block I/O Trace Replayers

June 2015, KCC

Kyuhwan Lee, Yong-Yeon Jo, Sang-Wook Kim

This paper analyzes the characteristics of the block I/O trace replayer used to evaluate the performance of SSD and verifies that the application can be replayed as if it were actually executed.
